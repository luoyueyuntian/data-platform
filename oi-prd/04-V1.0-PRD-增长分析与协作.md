# V1.0 PRD - 增长分析与协作

## 1. 版本目标

V1.0 的目标是把分析结果沉淀成可重复使用的增长资产。上线后，团队应能够基于分群、用户细查、路径分析、分布分析、间隔分析、虚拟事件、渠道链接和智能预警建立完整的增长协作闭环。

## 2. 用户价值

1. 运营可以快速找到目标用户群并持续追踪
2. 客服和成功团队可以查看单用户行为序列
3. 市场团队可以管理追踪链接并看推广质量
4. 分析师可以通过虚拟事件和高阶模型减少重复配置
5. 管理层可以对关键指标异常及时得到通知

## 3. In Scope

1. 用户细查
2. 用户行为序列
3. 分群管理
4. 路径分析
5. 分布分析
6. 间隔分析
7. 虚拟事件
8. 渠道链接管理
9. 智能预警第一版
10. 公共概览完整版
11. 更细的数据权限控制

## 4. Out of Scope

1. 标签体系
2. 用户画像全景视图完整版
3. 归因分析
4. 指标平台
5. 异常智能诊断高级版

## 5. 页面清单

| 页面 | 优先级 | 说明 |
| --- | --- | --- |
| 用户细查 | P0 | 精准查找用户与查看列表 |
| 用户详情 / 行为序列 | P0 | 查看单用户属性与事件轨迹 |
| 分群列表 | P0 | 创建、管理、导出分群 |

## 10. 版本完成定义

V1.0 完成后，平台不再只是“分析工具”，而是“增长团队能够反复使用的数据协作平台”。
