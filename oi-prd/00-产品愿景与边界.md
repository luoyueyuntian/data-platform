# 产品愿景与边界

## 1. 产品定位

OI Analytics Cloud 是一个面向 SaaS 场景的用户行为分析与增长运营平台，服务产品、运营、市场、分析师和研发团队，帮助企业完成以下闭环：

1. 采集多端行为与业务数据
2. 管理事件、属性、口径与数据质量
3. 对用户行为、属性、转化、留存、路径、分布进行自助分析
4. 将分析结果沉淀为分群、标签、概览和预警
5. 将分析结论用于增长、渠道优化和精细化运营

## 2. 产品核心价值

1. 降低数据分析门槛，让业务角色不依赖 SQL 完成大部分分析
2. 将“数据接入、治理、分析、运营”串成一条可持续复用的链路
3. 在 SaaS 模式下提供清晰的租户隔离、项目隔离、角色权限和资源共享机制
4. 支持从 MVP 起步，逐步扩展为具备 CDP 能力的增长平台

## 3. 核心用户角色

| 角色 | 核心诉求 | 典型动作 |
| --- | --- | --- |
| 租户管理员 | 创建组织、项目、成员、权限 | 管理账号、项目、角色、配额 |
| 项目管理员 | 管理项目内数据与分析资源 | 管理埋点、元数据、概览、预警 |
| 研发 / 数据接入工程师 | 完成 SDK 接入与数据校验 | 对接 SDK、调试上报、修复数据 |
| 产品经理 | 评估功能效果与关键流程 | 事件分析、漏斗、路径、留存 |
| 运营 | 做用户分群和活动效果复盘 | 分群、属性分析、概览、预警 |
| 市场 / 渠道 | 评估投放质量与归因结果 | 渠道链接、归因分析、分群 |
| 数据分析师 | 建立统一口径和高级洞察 | 虚拟事件、指标、标签、诊断 |
| 客服 / 成功团队 | 查询单个用户行为轨迹 | 用户细查、用户序列、用户画像 |

## 4. 产品对象模型

| 对象 | 定义 | 版本起点 |
| --- | --- | --- |
| 租户 Tenant | SaaS 组织级隔离单元 | V0.1 |
| 项目 Project | 数据接入与分析的业务空间 | V0.1 |
| 数据源 Data Source | SDK / API / Import 的接入来源 | V0.1 |
| 元事件 Event | 系统中可分析的标准行为事件 | V0.1 |
| 事件属性 Event Property | 附着于事件的维度与指标字段 | V0.1 |
| 用户 User | 受行为追踪的主体 | V0.1 |
| 用户属性 User Property | 用户级静态或动态属性 | V0.1 |
| 虚拟事件 Virtual Event | 由多个事件组合出的分析对象 | V1.0 |
| 分群 Segment | 由规则、导入或分析结果形成的用户集合 | V1.0 |
| 标签 Tag | 对用户的规则化分类结果 | V2.0 |
| 分析查询 Query | 一次分析模型配置与执行结果 | V0.2 |
| 概览 Dashboard | 多个分析卡片组成的看板 | V0.2 |
| 预警 Alert | 对指标异常或阈值触发的规则 | V1.0 |
| 渠道链接 Channel Link | 带追踪参数的推广链接 | V1.0 |
| 指标 Metric | 统一口径的业务指标定义 | V2.0 |

## 5. 顶层业务闭环

1. 研发完成多端埋点接入
2. 平台自动接收入库并做实时校验
3. 管理员维护元数据、字段显示名、类型和治理规则
4. 业务通过分析模型产出趋势、分布、转化和留存结论
5. 将结果沉淀为分群、标签、概览、预警
6. 运营与市场基于这些资产执行精细化动作
7. 团队持续迭代埋点与分析口径

## 7. 非功能要求总则

1. 默认支持 SaaS 多租户隔离
2. 默认所有分析请求都必须带租户与项目上下文
3. 默认所有资源都具备创建者、更新时间、权限范围、软删除能力
4. 默认所有关键操作可审计
5. 默认支持中英文显示名分离
6. 默认支持事件属性、用户属性的数据类型校验
7. 默认支持百万级用户、亿级事件规模下的分页、筛选与异步查询策略

## 8. 本产品不做的事情

1. 不在 V0.x 阶段提供完整 CDP 旅程编排
2. 不在 V1.0 阶段提供复杂机器学习建模平台
3. 不在当前文档中定义底层大数据基础设施实现细节
4. 不把 PRD 写成某一技术栈的代码设计文档，技术选型由开发阶段补充

## 9. 成功指标

| 维度 | 指标 |
| --- | --- |
| 接入效率 | 新项目接入首个事件的平均时长 |
| 数据质量 | 上报错误率、元数据冲突率、强校验拦截率 |
| 分析活跃 | 周活跃分析用户数、日均分析查询数 |
| 业务沉淀 | 分群数、概览数、预警数、共享次数 |
| 增长价值 | 关键漏斗提升、留存改善、投放 ROI 改善 |
| 平台可用性 | 查询成功率、P95 查询时延、告警送达率 |
